THE SPIN-OFF THAT SHOULD HAVE BEEN

Running 2015's first half, there was an announce on the Ionic Frameworks blog: The Ionic Jedi Hackster Hackathon.

Competing developers should build a Ionic App using the StarWars API, and Ionic (version 2 or later). I decided to play that game, and give me that required impulse to drive myself into learning Ionic and Typescript. Coming from an AngularJS background it didn't seem an easy task.

I coded day and night, for 2 days, the SW-Jokes app, a curated list of Star Wars jokes (some of them kind of funny). Got the 5th place, some stickers, a pen, a T-Shirt (I love that T-Shirt), a bottle opener.

Anyway, the app was incomplete. It required some extras not present in the original competition code. One of these extras was the Shryriiwook translation pipe. Searched in github for something similar to the wookie mode translation in the Star Wars API, and found some Ruby code.

I ended translating the Ruby code to an Angular pipe, that does the job. It's just a plain letter to pair of letters replacement, but most Shyriiwook translated text reads as Chewie's growls. But this was still incomplete. Some future features should include the capability of reading the jokes outloud.

Today I'm coding using Ionic 4, and distributing the app as a PWA (wrote a nice article on this).

Try it out, choose a male voice and set it with a low rate and pitch to achieve growling voices.
